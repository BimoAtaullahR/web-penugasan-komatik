lanjutan pengembangan dari penugasan pertama, penugasan kali ini akan mengimplementasikan materi dari apa yang telah kita pelajari pada learning lession "Frontend Development Best Practices". Teman-teman diharapkan untuk mengmbangan projek yang telah dibuat dengan menambahkan aspek aspek, seperti:

Ekspektasi Implementasi
Setiap project harus menunjukkan penerapan konsep berikut:
Scalable Folder Structure: Menggunakan struktur folder yang terorganisir, seperti feature-based structure dan atomic design.
Advanced State Management: Menerapkan local state, global state, dan server state sesuai kebutuhan aplikasi.
API Integration Best Practices: Menggunakan centralized API client, custom hooks, serta pengelolaan loading dan error state yang baik.
Performance Optimization: Menerapkan teknik optimasi seperti debouncing, lazy loading, memoization, atau code splitting.
Design System Implementation: Menggunakan design tokens, komponen reusable, serta konsistensi warna, spacing, dan typography.
Ekspektasi Penjelasan Video
Video tidak hanya menampilkan hasil akhir, tetapi juga menjelaskan bagaimana setiap best practice diterapkan pada project yang dibuat.

Contoh penjelasan:
“Pada project ini saya memilih studi kasus dashboard produk. Struktur project saya menggunakan feature-based folder structure sehingga setiap fitur seperti products, search, dan favorites terpisah dengan jelas. Untuk state management, saya menggunakan local state untuk form pencarian, global state untuk data favorit, dan server state menggunakan TanStack Query untuk pengambilan data dari API. Integrasi API saya lakukan melalui centralized API client dengan custom hooks agar kode lebih terstruktur dan mudah dikelola. Pada sisi performa, saya menerapkan debouncing pada fitur pencarian serta lazy loading pada beberapa komponen. Selain itu, saya menggunakan design system berupa reusable components dan design tokens agar tampilan aplikasi konsisten.”
