Berdasarkan materi modul Modern Frontend Framework (React & Next.js), setiap peserta
diminta membuat project sederhana yang memungkinkan terimplementasinya konsep utama
React dan Next.js, seperti: 
Component-Based Architecture 
State Management pada Component 
Mekanisme React (Virtual DOM, re-rendering, update UI) 
Client Side Rendering (CSR) 
Server Side Rendering (SSR) 
Next.js App Router 
Tugas wajib menggunakan Next.js App Router. 
Ketentuan Umum 
Gunakan Next.js App Router 
Pilih salah satu studi kasus yang telah disediakan 
Project dibuat secara individu 
Seluruh konsep utama yang telah dipelajari wajib diimplementasikan ke dalam
project 
Kreativitas desain, fitur, dan alur website dibebaskan kepada masing-masing peserta 
Fokus utama pada pemahaman konsep, bukan kompleksitas UI 
Pengumpulan dalam bentuk video presentasi 
Durasi video maksimal 10 menit 
Studi Kasus 
1. Dashboard Produk 
Buat website bertema dashboard produk yang menampilkan informasi produk, pencarian,
kategori, detail produk, atau fitur lain yang relevan. Kreativitas tampilan dan fitur dibebaskan. 

 2. Website Artikel / Blog 
Buat website bertema artikel atau blog yang menampilkan daftar artikel, halaman detail
artikel, interaksi pengguna, atau fitur lain yang relevan. Kreativitas tampilan dan fitur
dibebaskan. 

 3. Sistem Pemesanan Tiket / Event 
Buat website bertema pemesanan tiket atau event yang menampilkan daftar event, detail
event, pemesanan, ringkasan pesanan, atau fitur lain yang relevan. Kreativitas tampilan dan
fitur dibebaskan. 

Ekspektasi Penjelasan Video
Video tidak hanya menampilkan hasil akhir, tetapi juga menjelaskan bagaimana konsep
React dan Next.js diterapkan pada project yang dibuat. 
Contoh penjelasan: 
 “Pada project ini saya memilih studi kasus dashboard produk. Halaman utama
menampilkan daftar produk yang datanya saya render dari server sehingga saat
halaman dibuka konten langsung muncul. Lalu pada fitur pencarian saya
menggunakan state untuk menyimpan input pengguna. Ketika user mengetik,
state berubah dan React melakukan re-render pada bagian daftar produk saja.
Secara mekanisme, React membuat Virtual DOM baru lalu membandingkannya
dengan Virtual DOM sebelumnya, sehingga hanya elemen yang berubah yang
diperbarui ke DOM asli. Untuk navigasi detail produk saya menggunakan App
Router dari Next.js melalui struktur folder routing. Dengan begitu saya sudah
menerapkan konsep mekanisme react.”

Catatan. 
UI sederhana tidak menjadi masalah. Responsive menjadi penambahan

Penggunaan state

Perbedaan CSR dan SSR 
Cara kerja routing pada Next.js
